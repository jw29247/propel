# `propel setup`

Initialize `~/.propel/propel.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
propel setup
propel setup --workspace ~/.propel/workspace
```

To run the wizard via setup:

```bash
propel setup --wizard
```
